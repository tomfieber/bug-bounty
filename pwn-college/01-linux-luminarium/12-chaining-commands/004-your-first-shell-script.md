As you combine more and more commands to achieve complex effects, the length of the combined prompt quickly gets really annoying to deal with. When this happens, you can put these commands in a file, called a _shell script_, and run them by executing the file!

Same as last level, run `/challenge/pwn` and then `/challenge/college`, but this time in a shell script called `x.sh`, then run it with `bash`!

>[!tip]
>Put this in a file named x.sh


```sh
/challenge/pwn; /challenge/college
```
