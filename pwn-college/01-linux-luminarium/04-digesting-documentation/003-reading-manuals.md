The challenge in this level has a secret option that, when you use it, will cause the challenge to print the flag. You must learn this option through the man page for `challenge`!

```bash
# Bring up the man page
man challenge
```

![](../../../attachments/pwn-college/01-linux-luminarium/04-digesting-documentation/003-reading-manuals.md/Pasted%20image%2020250916161959.png)

Now we can run the command with the correct option to get the flag

```bash
/challenge/challenge --kbrgim 061
```
